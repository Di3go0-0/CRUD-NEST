# Api Task

## Description
The goal of this API is to demonstrate the usage of Nest.js, utilizing Prism as the ORM, and Swagger for documentation.

## Installation

```bash
$ npm install


## Running the app

```bash
# Development
$ npm run start

# Watch mode
$ npm run start:dev

# Production mode
$ npm run start:prod


## Test

```bash
# Unit tests
$ npm run test

# End-to-end tests
$ npm run test:e2e

# Test coverage
$ npm run test:cov
```
